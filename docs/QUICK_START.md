# WorthIt - Quick Start Guide

**Version**: 1.0.0
**Last Updated**: June 2026

---

## 5-Minute Overview

WorthIt is an AI-powered app that helps you decide **before** you spend money.

**Main Features**:
1. **"Should I Buy This?" Calculator** - Get AI advice in seconds
2. **Expense Tracking** - Log every purchase
3. **Budget Management** - Set limits by category
4. **Savings Goals** - Track progress toward what matters
5. **Financial Insights** - Understand your spending patterns

---

## Getting Started

### Step 1: Sign Up (2 minutes)
```
1. Go to worthit.app
2. Click "Sign Up"
3. Enter email and password
4. Or use "Sign in with Google"
```

### Step 2: Financial Profile (2 minutes)
```
1. Enter monthly income
2. Enter current savings
3. Enter total debts
4. Select currency (USD)
```

### Step 3: Choose AI Personality (1 minute)
```
Options:
- Friendly Mentor (supportive)
- Financial Advisor (analytical)
- Strict Mode (savings-focused)
- Student Mode (relatable)
- Future Self (perspective-shifting)
- Reality Check (honest)
```

### Step 4: Start Using App
```
Dashboard shows:
- Financial health score
- Spending breakdown
- Recent expenses
- Budget status
```

## Using "Should I Buy This?"

### The Process (30 seconds)

1. **Click** "AI Coach" button
2. **Enter** product name (e.g., "AirPods Pro")
3. **Enter** price ($249.99)
4. **Select** category (Electronics)
5. **Choose** Need or Want
6. **Select** emotional state
7. **Click** "Analyze"

### What You Get

```
Worth It Score: 38/100 ⚠️

Affordability: 82/100 ✓
Financial Impact: 25/100 ✗
Emotional Risk: 78/100 ✗
Goal Impact: 20/100 ✗

AI Coach Message:
"I'd pump the brakes here. You're stressed and 
considering an impulse purchase. Wait a week."

Alternatives:
- Refurbished AirPods: $199 (save $50)
- Budget option: $79
```

## FAQ

### Q: Is my data safe?
A: Yes. End-to-end encrypted, SOC 2 compliant, Firebase-backed.

### Q: What's the AI using?
A: Google Gemini API + financial algorithms + behavioral psychology.

### Q: Can I change my AI personality?
A: Yes, anytime in settings.

### Q: Does it work offline?
A: Partially. Expense logging works offline. AI coach needs internet.

### Q: Can I export my data?
A: Yes, from Settings → Export as CSV or PDF.

### Q: What if I disagree with the AI?
A: You decide! The AI is advice, not rules. Buy if you want.

### Q: Can I sync across devices?
A: Yes, automatically with your account.

### Q: What if I delete an expense?
A: It's gone. Budget and analytics will update automatically.

## Tips for Success

### 1. Log Everything
- Every coffee, every purchase
- Helps AI give better recommendations
- Track spending patterns

### 2. Set Realistic Budgets
- Based on your actual spending
- Adjust monthly as needed
- Tighter budgets = more savings

### 3. Use AI Coach
- Before major purchases ($50+)
- When you're emotional
- To understand spending triggers

### 4. Review Weekly
- Check budget status
- Celebrate wins
- Adjust strategies

### 5. Set Real Goals
- Save for something meaningful
- Break big goals into milestones
- Celebrate progress
